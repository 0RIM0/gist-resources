# このページの説明

ローカルファイルとテキストエリアを同期させます  

ファイルを保存するとテキストエリアに反映します  
テキストエリアを更新するとファイルに反映します  

テキストエリアの内容 (markdown) ファイルを HTML に変換し、出力先ファイルに保存します  
出力内容を iframe でプレビュー表示します  

# 使い方

適当な markdown ファイルを用意します  
面倒ならこのファイルをダウンロードして使えます  

右上の同期ボタンからソースにするローカルファイルと出力先ファイルを選択します  
終了ボタンで終了します  

## ヘッダーボタン

- 「？」
  - この markdown ファイルを開きます
- 「ソース表示切り替え」
  - テキストエリアの表示非表示を切り替えます
  - ローカルファイルを編集すればいいので不要ならテキストエリアを非表示にできます
- 「ソースファイルへ反映」
  - テキストエリア → ローカルファイル への反映を ON/OFF を切り替えます
  - OFF の場合は readonly になり、テキストエリアの編集自体ができなくなります
- 「同期」「終了」
  - ローカルファイルとテキストエリアの同期と HTML への変換の開始/終了をします
